# Blog API Project - Full Stack Blog Platform

A complete blog platform with separate backend API, public blog frontend, and author dashboard. Built with Node.js, Express, React, and PostgreSQL.

## 🚀 Live Demo

- **Public Blog**: [https://your-blog.vercel.app](https://blog-api-project-frontend1-blogkm.vercel.app/)
- **Author Dashboard**: [https://your-dashboard.vercel.app](https://blog-api-project-frontend2-auth.vercel.app)
- **Backend API**: [https://your-api.onrender.com](https://blog-api-project-backend-iixp.onrender.com/)

## 🛠 Tech Stack

### Backend

- **Runtime**: Node.js 18+
- **Framework**: Express.js
- **Database**: PostgreSQL (Neon Tech)
- **ORM**: Prisma
- **Authentication**: JWT (JSON Web Tokens)
- **Security**: bcryptjs, helmet, cors

### Frontend 1 - Public Blog

- **Framework**: Vanilla JavaScript
- **Styling**: Tailwind CSS
- **Features**: Post viewing, commenting

### Frontend 2 - Author Dashboard

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS
- **Rich Text Editor**: TipTap
- **Routing**: React Router v6
- **HTTP Client**: Axios

## 📦 Prerequisites

- Node.js 18+ installed
- PostgreSQL database (or Neon account)
- Git
- Vercel account (free)
- Render account (free)
